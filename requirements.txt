# STEP 1 — Install PyTorch before running pip install -r requirements.txt.
# PyTorch CUDA builds are not on PyPI. Pick the command for your hardware:
#
#   CUDA 11.8 (Linux server / GPU machine):
#   pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 --index-url https://download.pytorch.org/whl/cu118
#
#   CUDA 12.1:
#   pip install torch==2.0.1+cu121 torchvision==0.15.2+cu121 --index-url https://download.pytorch.org/whl/cu121
#
#   CPU only (Windows / Mac local testing, no GPU):
#   pip install torch==2.0.1 torchvision==0.15.2
#
# STEP 2 — Install GroundingDINO from the vendored submodule:
#   cd models/GroundingDINO && pip install -e . --no-build-isolation && cd ../..
#
# STEP 3 — Then install everything below:
#   pip install -r requirements.txt
#
# NOTE — pydensecrf is intentionally omitted. It has no Windows wheel and
# fails on Mac without a Cython build environment. Use the --no_crf flag
# when running evaluation scripts. Linux server users who want CRF can run:
#   pip install pydensecrf
